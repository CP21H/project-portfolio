dsi Library


Description


The dsi library is an attempt at a recreation of the C++ STL Library, meant for me to learn how the STL library is created, and more generally, the process of creating a library. The dsi library includes a set of classes dervied from the STL library, those classes are: algorithms.h, linked_list.h, priority_queue.h, queue.h, stack.h, and vector.h.


The program covers the following topics:

  • Algorithms
  • Linked Lists
  • Priority & Normal Queues
  • Stacks
  • Vectors

    • Outcome


    Learned the ins-and-outs of how to create a CMake file, templating, typenames, and pairing and linking libraries from library to project. Creating a library in general has been incredibly useful in advancing my knowledge.
