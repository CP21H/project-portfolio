Everything Arrays


Description


This project aims at consolidating the ways we can manipulate arrays into one program that provides easy reviewability.


The program covers the following topics:

  • Inserting Elements
  • Shifting Elements
  • Deleting Elements
  • Reversing an Array
  • Average of an Array
  • Max Value in an Array
  • Growing an Array
  • Shrinking an Array

    • Outcome


    Helped me review standard C Array manipulation without the use of the vector class / rebuilding functions of the vector class.
