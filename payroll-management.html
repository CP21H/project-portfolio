Payroll Management System


Description


The Payroll Management System served to get me accustomed to database writing in SQL given its large presence still today as one of the dominating languages / tools.


The database covers the following topics:

  • Company Info Table
  • Departments Table
  • Pay Grade Table
  • Employees Table
  • Payroll Table

    • Outcome


    Learned how to operate around the MySQL workbench, create databases with multiple tables, as well as stored procedures.


    The next step, and future idea for this project is to pair it with PHP or Node.js to create a website that can manipulate a SQL database through a web page.
