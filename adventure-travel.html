Adventure - Travel Designer


Description


The Adventure - Travel Designer website is the first in-depth and longer project I have worked on. I wanted to create a travel designer out of interest for travel, and so I tried to recreate the travel designer LuxTrips.co.


The program covers the following topics:

  • Immersive Home Page
  • Luxury Packages Page
  • Book With Us Page
  • Contact Page

    • Outcome


    Exponentially grew my knowledge and practice of HTML, CSS, and JS through creating a website that is visually on par with modern design patterns.


    Features Learned: Linear Gradient + Background Image pairing, ::-webkit-scrollbar, transforms, :checked, !important, transitions, -webkit backdrop-filters
