String Manip & Matrix Search


Description


This project consists of two programs, one handling string manipulation, and the other conducting operations on a matrix in C++. Through this project, I learned how a matrix can be represented, as a vector of vectors, and furthermore how one can search for a given word if this matrix is made of up vectors of chars.


The program covers the following topics:

  • String Creation, Manipulation, and Information Parsing
  • String Functions: erase(), replace(), compare(), swap(), find()
  • Matrix Creation: Vector of Vectors of Chars
  • Function called `find_in_matrix()` which finds a word in a matrix of chars vertically and horizontally

    • Outcome


    Learned the ins-and-outs of how to create a CMake file, templating, typenames, and pairing and linking libraries from library to project. Creating a library in general has been incredibly useful in advancing my knowledge.
