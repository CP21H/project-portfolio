C++ Review Program


Description


Starting the 2023 summer, this small C++ Review Program was made so that I could keep in practice what I had learned at university, as well as generally have a reference program I can use should I need to.


The program covers the following topics:

  • if-statements
  • while, do-while, for loops
  • Arrays
  • File I/O Stream
  • Classes
  • Dynamic Memory Allocation + Garbage Collection
  • Purely Virtual / Virtual

    • Outcome


    Helped me get back into programming in C++ by refreshing my knowledge of the language.
