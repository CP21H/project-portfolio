Password Generator & Strength Checker


Description


Password Generator and password strength checker program which rates the strength of a password based on certain criteria. Passwords generated are based on this criteria which is understood here as a rule set that is followed. The repository provides both the original implementation in C++ in the `csrc` file as well as a website link.


The rule set used for password generation and strength checking is listed in the README.md of the repository.

Outcome


Introductory program towards a knowledge of Cybersecurity. Exposed me to working with JavaScript more with translating the original source code created into JS so as to provide a Website for the user.
