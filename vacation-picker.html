Vacation Spot Picker


Description


This project asks the user a series of questions and depending on their answers, the program generates a few ideal locations for the user to vacation to. The program uses an array of strings to act as a quick database that can be iterated over without need for file input / output.


The program covers the following topics:

  • Culture Preference
  • Price Range (Functionality to be added later)
  • Beach, Mountain, Forest Preferences
  • Climate Preference

    • Outcome


    Helped me create a program that helps people decide where they want to go on vacation, answering a real world "problem" or instance.


    There is still much to gain from this project. In the future, I'd like to incorporate price range functionality and also create a rewrite of this project in JavaScript and HTML/CSS so that there is a visual for the user.
