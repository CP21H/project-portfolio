Pomodoro Timer


Description


Standard Pomodoro Timer which allows the user to study using the Pomodoro technique with a robust website allowing them features like: changing the background theming, changing the modes of time, being alerted when time is up, and users are provided a few recommended playlists.


This Pomodoro Timer has the following features:

  • Theme tab with various themes
  • Media tab with various playlists
  • Focus, Break, Long Break modes
  • Start, Stop, Reset


    • Outcome


    First project that was primarily JavaScript programming. I learned a lot about Date(), getHours(), getMinutes(), getSeconds(), set and clear interval.
