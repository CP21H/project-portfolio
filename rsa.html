RSA Encryption


Description


This RSA-Encryption program serves as an introductory program for me to familiarize myself with the application of cryptographic systems that I had learned in my Discrete Mathematics course in University. Encryption is originally implemented in C++ and then has been translated into JavaScript so as to provide a user-friendly website to demonstrate the encrpytion.


Original C++ source code can be found in the CSRC folder in the repository.

Outcome


Helped me grasp how cryptographic methods are translated from a purely mathematical basis to written code, helping me visualize these algorithms and understand them more thoroughly.
