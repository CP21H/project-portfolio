Bank Management System


Description


ATML-like program which allows the user to login, create an account, view their checkings and savings account, as well as deposit and withdraw money. The program uses recursive methods, the vector class, and for-each loops. Accounts were stored in a vector of account objects.


The program covers the following topics:

  • Start / Welcome Menu
  • Login Function
  • Create Account Function
  • Account Menu (Checkings/Savings)
  • Depost & Withdraw Functions

    • Outcome


    Refreshed myself with more intensive C++ programming. Introduced myself to and learned a few new libraries and concepts such as recursion and the vector class.
